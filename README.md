## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

* Python 3, An IDE 

* RPLY, llvmlite libraries installed

* Git

### Installation

1. Clone the repo
 ```sh
   git clone https://github.com/benymaxparsa/Compiler-Project.git
 ``` 
2. Write the code in input.nullr
3. Run the code :)
